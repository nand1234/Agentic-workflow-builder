# Jira Ticket Generation Prompt

## Instructions

Create a Jira ticket using the format below.

### Rules

* Include only **Title**, **Description**, and **Acceptance Criteria**
* Write **Acceptance Criteria in BDD style** (Given / When / Then)
* Do **not make assumptions**
* If any required detail is unclear or missing, **ask clarifying questions first and wait for answers before creating the ticket**
* Keep the content concise, clear, and technical

---

## Format

**Title:** <clear, action-oriented title>

**Description:**
<brief explanation of the change, including purpose and scope>

**Acceptance Criteria:**

* Given <context>
  When <action>
  Then <expected outcome>

* Given <context>
  When <action>
  Then <expected outcome>

---

## Task

<describe what needs to be done>

---

## Example

**Task:** Upgrade Cypress to version 15.13.1
