# Tickets

This directory contains implementation plans generated by /plan.

Each file is named after the feature slug:
- discount-codes.md
- user-auth.md
- password-reset.md

Files are created by /plan and updated by:
- /setup-worktrees — adds worktree paths
- /implement — updates ticket status
- /merge-to-cp — updates ticket status to merged-to-cp
- /rollback-ticket — resets ticket status to done
- /revise-plan — modifies todo tickets based on feedback
